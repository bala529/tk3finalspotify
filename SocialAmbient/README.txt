TO RUN Socialambientfacebook 

1) tk3final/SocialAmbient$ ant build
2) tk3final/SocialAmbient$ ant build facebook


TO RUN Socialambientspotify

1) tk3finalspotify/SocialAmbient$ ant build
2) tk3finalspotify/SocialAmbient$ ant build spotify



To RUN Socialambientlocalmusic

1) tk3finallocalmusic/SocialAmbient$ ant build
2) tk3finallocalmusic/SocialAmbient$ ant build local


Order of execution: 

1st possibility
1) Either Socialambientspotify or Socialambientlocalmusic has to be run in one system. 
Each one can be run separately on two different systems
2) Socialambientfacebook has to always run in a system. 

2nd possibility
1) Only Socialambientspotify or Socialambientlocalmusic can be run in the same system. 
2) Socialambientfacebook has to always run.
